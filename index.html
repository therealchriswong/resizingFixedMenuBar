The event of scrolling down adds a class small and removes the class small when you have scrolled to the top to the header element. The change in style triggers the transition in the header. A CSS3 transition provides a gradual change from one style to another for that particular element. You can specify the CSS property that you want to add an effect to and duration of the effect. The "All" keyword can be used for all properties to transition. The CSS class small, contains the styles for the header, img, ul when the menu is small.
